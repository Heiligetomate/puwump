# Puwump 

> [!WARNING]
> This probably does not work on Winslop (not tested)

## General info 

Puwump is a local desktop workout app. 

Saving data is achieved with a sqlite database.

## Features 

1. Create, Delete exercises 
2. Create, Delete ingredients 
3. Create, Delete workout plans 
4. Create, Delete meals 
5. Add exercises to workout plans 
6. Add ingredients to meals 
7. Workout 
8. Eat 
9. Log finished workouts (just in the db right now)
10. Log finished meals   (just in the db right now
