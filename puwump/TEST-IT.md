# Try it out 

## Step by step guide 

1. Have rust installed ([install here](https://rust-lang.org/tools/install/))
2. Go in your command line 
3. Go to the location you want the repository to be 
4. clone the repo: `git clone https://github.com/Heiligetomate/puwump`
5. go in the directory: `cd puwump/puwump`
6. run the program: `cargo run` or build the release: `cargo build --release`
